# Wyze Events Viewer

Unofficial visualizer for Wyze event-group data. Nothing leaves your browser — you paste a JSON snippet, the page renders a timeline of your events.

## Use it

Live: https://rokey-z.github.io/wyze-events-viewer/

1. Open <https://lumen.wyzecam.com/events> and sign in.
2. Tap the **?** button in the top-right and copy the producer snippet.
3. Paste it into the Wyze tab's DevTools Console, press Enter, wait for the "Copied N groups" alert.
4. Come back to the viewer and tap **Load from clipboard**.

Mobile users: use the bookmarklet link inside the Help modal.
